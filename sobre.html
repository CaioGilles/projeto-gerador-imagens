Sobre o Projeto

Este site foi criado com o objetivo de oferecer aos usuários uma forma simples e prática de gerar imagens utilizando inteligência artificial. A ideia nasceu da curiosidade em explorar o potencial criativo das IAs e torná-lo acessível para qualquer pessoa.

Além de ser uma ferramenta útil, este projeto também é uma maneira de aprender e aplicar conhecimentos em HTML, CSS e programação web.
